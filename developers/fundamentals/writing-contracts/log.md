---

You can view further documentation into [Blockchain.require()](https://avm-api.aion.network/avm/blockchain#log%28byte%5B%5D%29) on the [AVM API website](https://avm-api.aion.network/).

## Example Contract

You can use this contract as template.

```java
package aion;

import avm.Blockchain;
import org.aion.avm.tooling.abi.Callable;
import avm.Address;

public class EventExample {

    public static void emitEvent() {
        String eventTopic = "Emit Event"; // topic
        Address eventCaller = Blockchain.getCaller(); // data
        Blockchain.log(eventTopic.getBytes(), eventCaller.unwrap());
    }
}
```
